This project is a simple yet fun **balloon popping game** developed for Android devices. Players tap on rising balloons on the screen to pop them. As the game progresses, the difficulty increases: balloons rise faster. When the time is up, all remaining balloons on the screen automatically pop, and the game ends.

ğŸ® Game Features

- **2 Scenes (Screens):
  - **Start screen**: Starts the game with a "Start" button.
  - **Game screen**: A 60-second timer begins and the gameplay starts.

- Balloons:
  - Continuously rise from the bottom of the screen.
  - Players can tap the balloons to pop them.
  - Each popped balloon increases the score counter by one.
  - As time progresses, the rising speed of the balloons increases (difficulty ramps up).

- Score Tracking:
  - The number of popped balloons is shown in a corner of the screen.

- Game Over:
  - After 60 seconds, the game ends.
  - All remaining balloons on the screen automatically pop.
